dbmast.ru
***********
Небольшая инструкция по применению php-обработчика всплывающей формы контактов:

http://dbmast.ru/sozdaem-vsplyvayushhuyu-kontaktnuyu-formu-dlya-sajta

В архиве с формой находится папка "mail", в которой размещены два файла
1. config_mail.php - файл конфигураций(настроек) и взаимодействия с php-обработчиком формы. В коде файла прописаны комментарии к его основным раздела, так что не потеряетесь))).
 
2. Второй файл в папке "mail" непосредственно сам рhp-обработчик формы, файл sendletter.php, который будет собирать данные с контактной формы, компоновать их в сообщение и отправлять по электронной почте указанной в файле config_mail.php. 
Для того чтобы прикрутить обработчик к форме тег <form> должен иметь 2 дополнительных атрибута:

* action="mail/sendletter.php" - определяет куда отправлять данные из контактной формы после отправки письма, т.е в php-обработчик sendletter.php...

* method=»post» — определяет, как отправить данные из контактной формы в файл, заданный атрибутом action. Теги <input> и <textarea> должны иметь атрибут "name" с уникальным идентификатором. Этот атрибут используется для идентификации данных контактной формы после отправки письма.

Следует также отметить 1 дополнительный элемент формы, который используются в качестве кнопки Отправить, елемент должен быть определен тегом type="submit".

<input type="submit" name="send" value="Отправить" class="send-message">

Комментарии в коде обработчика помогут вам разобраться.
